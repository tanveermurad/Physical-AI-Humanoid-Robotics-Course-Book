INSTRUCTIONS:
1. Open each website below
2. Copy the credential
3. Paste it in the space provided
4. After collecting all 5, copy everything to deployment/.env

================================================================================

1. RENDER API KEY
   Website: https://dashboard.render.com/u/settings#api-keys
   Steps: Sign up → Go to link above → Create API Key → Copy

   Paste here:
   RENDER_API_KEY=


================================================================================

2. NEON DATABASE URL
   Website: https://console.neon.tech
   Steps: Sign up → Create Project → Copy "Connection String"

   Paste here (should start with postgresql://):
   NEON_DATABASE_URL=


================================================================================

3. QDRANT URL
   Website: https://cloud.qdrant.io
   Steps: Sign up → Create Cluster (Free) → Copy "Cluster URL"

   Paste here (should start with https://):
   QDRANT_URL=


================================================================================

4. QDRANT API KEY
   Same website: https://cloud.qdrant.io
   Steps: Go to API Keys → Create → Copy

   Paste here:
   QDRANT_API_KEY=


================================================================================

5. OPENAI API KEY
   Website: https://platform.openai.com/api-keys
   Steps: Sign up → Create new secret key → Copy

   Paste here (should start with sk-proj- or sk-):
   OPENAI_API_KEY=


================================================================================

6. GITHUB TOKEN
   Website: https://github.com/settings/tokens
   Steps: Generate new token (classic) → Check "repo" → Generate → Copy

   Paste here (should start with ghp_):
   GITHUB_TOKEN=


================================================================================

✅ AFTER COLLECTING ALL CREDENTIALS:

1. Open: deployment/.env.example
2. Copy everything to: deployment/.env
3. Paste your credentials from above into deployment/.env
4. Save the file
5. Run: python deployment/deploy_all.py
